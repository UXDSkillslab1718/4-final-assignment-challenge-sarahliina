Interaction Design

Wireframe

Interaction Design is a course that provides us with knowledge about user centred design techniques. These will help us to define relevant requirements with the intend to create a user-friendly application. We learned to translate user needs in to system requirements and to design an information structure that will fit these requirements to facilitate a good user - system interaction. As final project for this course we developed and created a wireframe for a fictional cinema company - UK Cinemas.
